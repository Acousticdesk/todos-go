# TODO app CLI using Go

## Pre-requisites
- Install Go

## Run the App
`go run *.go`

## About the App
The application allows you to create your todo list using CLI

### Available commands
| Command    | Description                         |
| ---------- | ----------------------------------- |
| help       | output all the available commands   |
| create     | create a new todo                   |
| done       | mark a todo as completed            |
